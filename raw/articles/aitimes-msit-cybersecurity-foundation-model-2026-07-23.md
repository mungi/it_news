# 과기부 ‘사이버 보안 특화 AI 파운데이션 모델 개발’ 사업 공모

## Source facts
- 과학기술정보통신부와 NIPA가 2026년 7월 23일 국내 사이버 보안 특화 AI 파운데이션 모델 개발 공모에 착수함.
- 국내 AI·사이버 보안 기업, 대학, 연구기관이 컨소시엄으로 참여할 수 있고 주관기관은 국내 기업·기관으로 제한됨.
- 공모 마감은 8월 21일이며, 선정 컨소시엄에는 B200 GPU 256장(32노드)을 총 10개월 지원하는 계획.
- 지원 5개월 차에 중간 성과 모델을 공개하고 평가 결과로 이후 지원 여부를 결정하는 단계형 구조.
- 참여팀은 모델 개발 방법론, 모델 기반 서비스 실증, 데이터 처리·정보보호·유지보수 등 지속 가능 운영성을 제안해야 함.
- 멀티 에이전트 방식 서비스 연계와 오픈소스 지향이 언급됨.

## Source boundary
- 모델 크기, 학습 데이터, benchmark, 라이선스, 공개 범위, 배포 환경은 기사에서 확정되지 않음.
- GPU 지원·공모 목표는 특정 SOC·침해대응 환경의 모델 성능, 보안성, 운영 SLA를 보증하지 않음.
- 보안 자동화에 연결하는 경우 tool 권한, 승인, 감사·rollback 경계는 도입 조직이 별도 검증할 사항.
