# Amazon EC2 Dedicated Hosts, AMD SEV-SNP 지원으로 기밀 컴퓨팅 선택지 확대

- 출처: AWS What’s New
- 게시: 2026-07-03 02:00 KST
- 카테고리: Cloud

## 요약

핵심: EC2 Dedicated Hosts, AMD SEV-SNP 지원 — 전용 호스트 환경의 메모리 암호화·무결성 보호 강화

## 상세

변화: 규제·멀티테넌시 민감 워크로드에서 기밀 컴퓨팅 옵션이 인스턴스 수준을 넘어 전용 호스트 운영 모델로 확장

## 왜 중요한가

금융·공공·민감 데이터 워크로드는 클라우드 사용 시 하드웨어 기반 격리와 감사 가능성을 요구합니다. 전용 호스트와 SEV-SNP 조합은 격리 설계 선택지를 늘립니다.

## 시사점

기밀 컴퓨팅 PoC는 성능 오버헤드, attestation, 키 관리, 이미지 호환성, 운영 런북, 규제 감사 증적을 함께 검토해야 합니다.
