# Grok 4.5 공개, 코딩·에이전트 작업 성능과 출력 토큰 효율 전면 강조

- AI Times와 GeekNews 정리 기준 SpaceXAI가 코딩·에이전트 작업용 Grok 4.5를 공개함.
- 공개 수치에는 Terminal Bench 2.1 83.3%, SWE Bench Pro 64.7%, DeepSWE 1.0 62.0%, DeepSWE 1.1 53%, 80 TPS, SWE Bench Pro 작업당 평균 출력 토큰 15,954개가 포함됨.
- Grok Build, Cursor 플랜, SpaceXAI 콘솔에서 제공되고 EU 제공은 7월 중순 예정으로 설명됨.
- 운영 의미: 코딩 모델 평가는 pass rate뿐 아니라 작업당 토큰, latency, tool-call 안정성, IDE 통합 보안 조건까지 포함 필요.
