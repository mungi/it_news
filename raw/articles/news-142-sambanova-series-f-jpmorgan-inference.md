## 핵심 요약
- SambaNova가 110억달러 가치로 10억달러 규모 Series F 1차 조달을 완료함.
- JP모건체이스가 SN40L과 차세대 SN50으로 사내 데이터센터 온프레미스 AI 추론 인프라를 구축할 계획.
- SambaNova는 학습은 NVIDIA GPU, 응답 생성·decode는 자체 칩이 처리하는 보완 전략을 강조함.
- 회사는 SN40/SN50의 LLM decode 성능이 NVIDIA GPU보다 5~10배 높다고 주장함.

## 운영 메모
- 추론 ASIC PoC는 tokens/sec, p95 latency, context length, batch 패턴, compiler/runtime 성숙도, 장애 재기동, 보안 패치와 공급망 리드타임을 함께 측정 필요.
- 금융·정부 고객은 public API 비용뿐 아니라 데이터 residency와 auditability를 전용 추론 인프라 평가 기준으로 설정 필요.
