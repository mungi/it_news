# ScarfBench: Benchmarking AI Agents for Enterprise Java Framework Migration

## Korean briefing summary
Hugging Face Blog는 IBM Research의 ScarfBench를 소개했습니다. 엔터프라이즈 Java 프레임워크 마이그레이션 작업에서 AI 에이전트 성능을 평가하기 위한 벤치마크입니다.

## Detail
기업 시스템 현대화는 단순 코드 생성보다 대규모 의존성, 테스트, 프레임워크 차이를 다룹니다. ScarfBench는 AI 에이전트가 실제 레거시 전환 작업에서 얼마나 쓸모 있는지 측정하려는 흐름을 보여줍니다.

## Why it matters
AI 코딩 에이전트 평가가 장난감 문제에서 실제 엔터프라이즈 마이그레이션으로 이동하고 있습니다.

## Engineering implication
조직은 자사 코드베이스에 맞는 migration benchmark와 회귀 테스트를 만들어 도구를 평가해야 합니다.
