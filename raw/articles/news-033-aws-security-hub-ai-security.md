# AWS Security Hub CSPM launches AI Security Best Practices standard with 31 automated controls

## Source facts
- AWS What’s New announced the AI Security Best Practices standard for AWS Security Hub CSPM.
- The standard contains 31 automated security controls.
- It evaluates Amazon Bedrock, Amazon Bedrock AgentCore, and Amazon SageMaker workloads against recommended security configurations.
- Covered domains include network isolation, encryption at rest and in transit, VPC placement, KMS key usage, private container registry requirements, and authorization controls.
- AWS states the standard is available in all AWS Regions where Security Hub CSPM is available, including AWS GovCloud (US) and China Regions.
