# Cloudflare Drop

- 원 출처: https://www.cloudflare.com/drop/
- 핵심: HTML/CSS/JS 정적 사이트 폴더나 ZIP을 업로드하면 Cloudflare 글로벌 네트워크에서 몇 초 안에 확인하는 서비스.
- 용도: 정식 CI/CD 배포 전 미리보기, 프로토타입 공유, 문서 사이트 임시 확인.
- 논점: HN/GeekNews 댓글에서 콘텐츠 라이선스 조항, 악성 정적 사이트 호스팅 마찰 감소, 내부 자료 업로드 통제 문제가 함께 제기.
- 시사점: 플랫폼팀은 임시 preview 배포 도구 사용 기준, 업로드 금지 데이터, 만료·삭제 절차, 감사 로그 대안을 정의 필요.
