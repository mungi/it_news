# AWS BCM Cost Efficiency 위젯: 비용 효율 추세를 대시보드에서 통합 관측

- AWS Billing and Cost Management Dashboards에 Cost Efficiency 위젯 추가
- 비용 효율 점수를 시간대별로 표시하고 account·region·전체 AWS 환경 범위로 필터링 가능
- Cost Explorer, Budgets, Savings Plans·Reserved Instances coverage/utilization report와 같은 dashboard에서 지출·약정·최적화 성과를 함께 확인 가능
- Cost Optimization Hub 콘솔 권고로 직접 연결되며 dashboard export, 예약 이메일, CSV·PDF, cross-account dashboard sharing에 포함
- AWS 발표 기준 모든 상용 리전에서 추가 비용 없이 제공

## 확인한 범위와 운영 경계

- 원문은 위젯의 점수 산식, 권고 절감액 정확도, 조직별 기준선은 설명하지 않음
- FinOps review에서는 efficiency trend와 commitment utilization, budget variance, action owner·기한을 같은 change evidence로 연결 필요
- cross-account sharing과 export 사용 시 IAM 접근 권한 및 비용 데이터 수신자 범위를 별도 검토 필요
