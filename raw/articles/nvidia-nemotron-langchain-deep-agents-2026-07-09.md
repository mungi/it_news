# NVIDIA Nemotron + LangChain Deep Agents

- LangChain Deep Agents harness를 Nemotron 3 Ultra에 맞춰 조정해 오픈 모델 최고 정확도와 closed model 상위권 business task parity를 제시함.
- NVIDIA는 실행당 추론 비용이 leading closed models 대비 10분의 1이라고 주장함.
- 개선은 모델 재학습이 아니라 system prompt, tool description, middleware 조정에서 발생함.
- NemoClaw for LangChain Deep Agents는 LangChain Deep Agents Code, Nemotron 3 Ultra tuned profile, NVIDIA OpenShell secure runtime을 묶는 참조 blueprint임.
- Baseten, Crusoe Cloud, DeepInfra, Fireworks, Nebius, Together AI를 통해 hosted production 경로 제공.
