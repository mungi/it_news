# Amazon CloudWatch announces coding agent insights

## Source facts
- Published: 2026-07-20 18:51 KST (AWS What’s New RSS `pubDate`: 2026-07-20 09:51 UTC)
- AWS announced CloudWatch Coding Agent Insights for visibility into coding-agent value.
- Claude Code integrates through Claude apps gateway for AWS without additional instrumentation; AWS also lists Codex and GitHub Copilot as supported coding agents.
- The dashboard combines OpenTelemetry metrics emitted by coding agents with CloudWatch operational data.
- AWS examples include spend trends, token billing alerts, commit throughput, pull request velocity, and model cost-to-output comparison.
- Available in AWS commercial regions except Middle East (UAE), Middle East (Bahrain), and Israel (Tel Aviv); standard CloudWatch OpenTelemetry metric ingestion pricing applies.

## Source boundary
- AWS does not claim that agent telemetry proves causal productivity gains, code quality, or a universal ROI.
- Metric fields, ingestion volume, retention, access control, and organization-specific costs require implementation verification.
