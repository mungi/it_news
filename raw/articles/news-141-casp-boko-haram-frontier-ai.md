## 핵심 요약
- CASP와 GeekNews GN+ 요약은 전 Boko Haram 구성원 27명, 대면 인터뷰 57회를 근거로 프런티어 AI의 조직적 활용 정황을 정리함.
- ISWAP·JAS가 ChatGPT, Claude, Gemini, Grok, Meta AI, DeepSeek를 무기 정비, 폭발 장치 설계, 전술 계획, 작전 보안, 물류 문제 해결에 활용했다는 증언 포함.
- 전담 AI 조직, 유료 구독, VPN·암호화 소프트웨어, 안전장치 우회 교육이 결합된 조직적 abuse model 정황.
- 실제 작전 성능 향상은 독립 입증되지 않았고 전직 구성원 자기보고 한계 존재.

## 운영 메모
- AI provider abuse 대응은 단일 프롬프트 필터보다 계정 군집, 결제·VPN 패턴, cross-model 재시도, 고위험 질의 시퀀스 분석 필요.
- 기업 조달팀은 high-risk abuse handling, audit evidence, provider escalation SLA를 API 공급자 평가 항목으로 포함 필요.
