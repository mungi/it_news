# OpenAI Codex·ChatGPT Work 활성 사용자 700만명

- AI타임스가 Codex 책임자 티보 소티오의 7월 13일 X 공개를 인용해 Codex와 ChatGPT Work의 합산 활성 사용자가 **700만명**에 도달했다고 보도함
- 시계열: Codex는 2월 정식 출시 후 3월 200만명, 4월 300만명, 6월 2일 500만명 수준으로 늘었고 이후 약 40일 만에 합산 700만명 수치 공개
- 기능: 오픈AI가 기념 기능으로 계정별 주간 사용량을 한 번 초기화하는 Banked Reset을 추가했다는 보도
- 운영: 사용자 수는 Codex 단독·유료 전환·지역·업종·유지율로 분해돼 공개되지 않아 서비스 용량·매출·단일 모델 효과로 일반화할 수 없는 조건

## 엔지니어링 해석 경계

- 700만은 Codex와 ChatGPT Work의 합산 활성 사용자 수치이며, Coding Agent 단독의 일간 활성 사용자·작업량·토큰량 지표가 아님
- GPT-5.6 출시 뒤 증가세가 빨라졌다는 분석은 기사 내 정황이며 단일 모델 출시의 인과관계·성능 효과를 입증하는 자료는 아님
- 조직 도입 판단은 공급자 성장 수치보다 사용자·작업별 quota, latency, retry, fallback, 권한·보존 조건을 자사 telemetry로 검증할 필요
