# 테트라멤·SK하이닉스 인메모리 컴퓨팅 SoC

- 핵심: 테트라멤과 SK하이닉스가 Analog In-Memory Computing 협력 결과로 멤리스터 기반 AI SoC 연구 성과 공개.
- 논문: “A Memristor-based In-Memory Computing SoC with Efficient Depthwise Convolution”이 Advanced Intelligent Systems 게재 및 표지 논문 선정.
- 기술: AI 모델 가중치가 저장된 위치에서 연산을 수행해 프로세서-메모리 데이터 이동, 전력, 지연, 발열 병목 완화 목표.
- 시사점: 추론 인프라 평가는 TOPS뿐 아니라 데이터 이동 에너지, compiler/runtime, 모델별 연산 매핑, 공급 안정성까지 포함 필요.
