# Tencent Hy3 공개, Apache 2.0 오픈소스 모델로 에이전트·업무 작업 비용 경쟁 가세

- 출처: https://news.hada.io/topic?id=31290
- 발행(KST): 2026-07-10 09:56
- 분류: AI / Global

## 요약

핵심: Tencent Hy3는 Apache 2.0 공개 모델로 270명 전문가 블라인드 평가 2.67/4, GLM-5.1 2.51/4를 제시하며 업무·에이전트 작업 성능을 강조. 비용: API 가격은 100만 토큰당 입력 1 RMB, 출력 4 RMB, 캐시 입력 0.25 RMB로 공개되어 모델 라우팅과 추론 단가 비교 대상 추가.

## 상세

핵심: Hy3는 제품 피드백과 후훈련 데이터를 반영한 Tencent의 오픈소스 모델 공개이며, 도구 호출·긴 컨텍스트·다회전 의도 추적 개선을 전면에 둠. 영향: Apache 2.0과 낮은 API 단가가 결합되면서 기업 모델 게이트웨이는 중국계 오픈 모델까지 품질·지연·비용 실험 매트릭스에 포함 필요.

### 무슨 일이 있었나
- 핵심: Tencent가 Hy3를 Apache 2.0 라이선스의 오픈소스 모델로 공개
- 맥락: 4월 말 Hy3 preview 이후 50개 이상 제품 피드백과 고품질 후훈련 데이터를 반영
- 배포: GitHub, Hugging Face, ModelScope, AtomGit에 모델 공개

### 본문 핵심 포인트
- 평가: 270명 전문가 실제 업무 기반 블라인드 평가에서 Hy3 2.67/4, GLM-5.1 2.51/4 기록
- 강점: 프론트엔드 개발, 데이터 및 스토리지, CI/CD 작업에서 상대 우위 확인
- 신뢰성: 내부 실제 시나리오 평가에서 환각률 12.5%에서 5.4%로 하락
- 오류율: 상식 오류율 25.4%에서 12.7%, 내부 다회전 테스트 이슈율 17.4%에서 7.9%로 하락
- 효율: WorkBuddy 내부 테스트에서 작업 성공률 72%에서 90%로 상승하고 평균 완료 시간 34% 감소
- 가격: API는 100만 토큰당 입력 1 RMB, 출력 4 RMB, 캐시 입력 0.25 RMB로 제시

### 왜 중요한가
- 경쟁: 오픈 모델이 에이전트 작업 성공률과 토큰 효율을 내세우면 폐쇄형 프런티어 모델의 기본 라우팅 지위 약화 가능성
- 라이선스: Apache 2.0 공개는 기업의 자체 호스팅, 파인튜닝, 감사 가능성 검토를 쉽게 만드는 조건
- 제약: 벤치마크와 업체 내부 평가는 운영 환경의 데이터 보안, 지연 시간, 장애 대응 능력을 대체하지 못함

### 시사점
- 팀 액션: 모델 게이트웨이에 Hy3 실험 슬롯을 만들고 업무 유형별 성공률·토큰 사용량·출력 품질을 기존 모델과 비교 필요
- 운영 과제: 중국계 모델 사용 시 데이터 거버넌스, 지역 규제, 공급망 신뢰성, 사내 보안 심사 기준을 별도 체크 필요
