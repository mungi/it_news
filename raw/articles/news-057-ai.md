## 요약

## 상세
핵심: 중국 당국, 최고 성능 AI 모델의 해외 접근 제한 검토
변화: 공개·미공개 차세대 모델과 오픈웨이트 모델까지 국가 전략 자산 관리 대상으로 논의
영향: 글로벌 AI 도입팀은 중국 모델 API·오픈웨이트 접근성, 라이선스, 배포 지역 제약을 공급망 리스크로 반영 필요
시사점: 모델 라우팅 전략은 가격·성능뿐 아니라 국가별 접근 통제, 데이터 처리 위치, 대체 모델 fallback을 포함 필요

## 왜 중요한가
중국 모델은 가격 경쟁력과 오픈웨이트 선택지로 해외 채택이 늘고 있음. 접근 제한이 현실화되면 멀티모델 전략과 AI 공급망이 미국·중국 규제 축으로 더 분리될 가능성 있음

## 시사점
모델 게이트웨이는 중국계 모델 의존도, 지역별 API 접근성, 오픈웨이트 재배포 조건, fallback 모델, 데이터 국외 이전 제한을 배포 체크리스트에 포함 필요
