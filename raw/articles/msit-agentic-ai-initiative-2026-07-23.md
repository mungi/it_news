# 과기정통부, 에이전틱 AI 이니셔티브 발표: 안전·개방형 실행 기반·수요 도입 3대 정책 축 제시

인공지능신문이 보도한 과기정통부 2026-07-23 에이전틱 AI 이니셔티브 발표을 바탕으로 작성. 3대 전략은 안전·신뢰, 개방형 실행 기반, 수요 중심 도입이며, 연내 가이드라인·agent identity·책임 소재 검토가 언급됨. 법적 의무, 기술 표준, 예산, 세부 공모와 시행 일정은 기사에서 확정되지 않은 범위임.

## 핵심 사실

- 정책 축: 안전·신뢰 선도, 개방형 실행 기반 조성, 수요 중심 도입 촉진의 **3대 추진 전략**
- 발행: 기사 입력 시각 **2026-07-23 13:04 KST** 확인
- 가이드라인: 에이전틱 AI 안전·신뢰 가이드라인을 연내 마련한다는 정책 방향
- 책임성: 에이전틱 AI 얼라이언스를 중심으로 agent identity와 행위 책임 소재 검토 계획
- 경계: 의무 요건·인증 기준·적용 대상·제재 수단은 기사만으로 확정 불가
- 도구: 생활 밀착형·고수요 agent service와 tool 개발 지원 계획
- 실행: multi-agent 협업·tool use·반복 호출의 높은 연산 요구에 대응하는 token 효율 실행 운영 기술 추진
- 모델: 국산 모델의 agentic performance 강화를 위한 GPU·agentic data 지원 계획
