# AWS Network Firewall now supports container attribute-based inspection for Amazon EKS and Amazon ECS

## Source
- Source: AWS What’s New
- URL: https://aws.amazon.com/about-aws/whats-new/2026/06/aws-network-firewall-container-attributes-referencing
- Published KST: 2026-07-01 02:00
- Ingested KST: 2026-07-06 10:02

## Korean briefing summary
AWS What’s New는 AWS Network Firewall이 Amazon EKS와 ECS의 컨테이너 속성을 참조하는 규칙을 지원한다고 발표했습니다. 컨테이너화된 워크로드와 생성형 AI 애플리케이션 보안을 단순화하는 기능으로 소개됐습니다.

## Detail
동적으로 변하는 컨테이너 환경에서는 IP 기반 정책만으로 워크로드 정체성을 표현하기 어렵습니다. 컨테이너 속성 기반 검사는 서비스·태그·메타데이터 중심의 네트워크 보안 정책에 가깝습니다.

## Why it matters
클라우드 네이티브 보안 정책이 인프라 주소가 아니라 워크로드 속성 중심으로 이동하고 있습니다.

## Engineering implication
EKS/ECS 네트워크 정책과 Firewall 규칙을 애플리케이션 메타데이터 기준으로 정리할 수 있습니다.
